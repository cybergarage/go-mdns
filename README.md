go-mdns is a Go library for Multicast DNS (mDNS) and DNS Service Discovery (DNS-SD) as defined in [RFC 6762](https://www.rfc-editor.org/rfc/rfc6762) and [RFC 6763](https://www.rfc-editor.org/rfc/rfc6763).
